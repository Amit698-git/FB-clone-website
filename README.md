# 📘 Facebook Clone Website

A **Facebook Clone Website** built using **HTML** and **CSS** as part of my Git and GitHub learning journey. This project was developed during a Git tutorial to practice version control while creating a real-world front-end application.

The project replicates the basic layout and design of Facebook, including the navigation bar, sidebar, stories section, post feed, and contacts panel. Along with improving my front-end development skills, this project gave me hands-on experience with Git workflows such as branching, merging, committing changes, and working with GitHub repositories.
